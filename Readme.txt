Avaliação Sprint 2 - Node
•	API serve para cadastrar estados com os seguintes parâmetros: nome, região, população, capital e área. O parâmetro id é gerado automaticamente a cada inserção.
•	Enviando a url http://localhost:3000/states via GET obtém-se a lista de todos os estados cadastrados.
•	Enviando a url http://localhost:3000/states/id via GET obtém-se os dados cadastrados com a id que foi passada.
•	Enviando a url http://localhost:3000/states via POST juntamente com os parâmetros, o cliente consegue cadastrar um novo estado.
•	Enviando a url http://localhost:3000/states via PUT juntamente com os parâmetros, o cliente consegue editar um cadastro já realizado.
•	Enviando a url http://localhost:3000/states/id via DELETE,  o cliente consegue apagar um cadastro cuja id foi passada.
